**Circuit File Generator (.ASC & .CIR)**

A simple tool that automatically generates circuit files based on user input values.

Run the program, enter the required parameters, select the circuit type, and the tool will generate:

.asc file → ready to open in LTspice schematic editor
.cir file → SPICE netlist that can be simulated or modified

This makes it easy to quickly create simulation-ready circuits without manually drawing schematics.

Features
Generate LTspice compatible .asc schematic files
Generate .cir SPICE netlist files
Supports multiple circuit configurations
Custom component values through user input
Fast and reproducible circuit creation
Useful for students, researchers, and engineers
How It Works
Run the program
Enter the required input values (example: resistance, capacitance, voltage, frequency, etc.)
Select the desired circuit configuration
The program automatically generates:
circuit_name.asc
circuit_name.cir
Open the .asc file in LTspice or simulate using the .cir netlist
